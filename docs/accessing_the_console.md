# Accessing The Console

Getting the console to show is simple.
The main executable, `main.pyw` uses the `.pyw` extension to hide the console.
By going to the terminal and running it with Python shows the console.

Example:
```bash
python3 main.pyw
```

## Note for Windows users:
Make sure you are using the python3/py command, otherwise it will run as normal.

Bad Example:
```powershell
"C:/Users/User/ABS-Engine/run.pyw"
```

*Good* Example:
```powershell
py C:/Users/User/ABS-Engine/run.pyw
```
